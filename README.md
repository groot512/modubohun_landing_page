# 모두보훈 랜딩페이지

## 개요
국가유공자와 그 유가족을 위한 모바일 신분증 기반 할인 서비스 '모두보훈'의 공식 랜딩페이지입니다.

## 파일 구성
- index.html: 메인 HTML 파일
- style.css: CSS 스타일시트
- script.js: JavaScript 파일

## 주요 기능
✅ HTML5 시맨틱 구조
✅ 반응형 디자인 (모바일 퍼스트)
✅ Schema.org 구조화된 데이터 마크업
✅ SEO 최적화 메타 태그
✅ Open Graph & Twitter Card 지원
✅ 접근성 향상 기능
✅ 부드러운 애니메이션
✅ 모바일 최적화

## 브라우저 지원
- Chrome (최신 버전)
- Firefox (최신 버전)
- Safari (최신 버전)
- Edge (최신 버전)
- 모바일 브라우저

## 사용 방법
1. 모든 파일을 웹 서버에 업로드
2. index.html을 브라우저에서 열기
3. 반응형 디자인으로 모든 기기에서 최적화된 화면 제공

## 기술 스택
- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript (ES6+)
- JSON-LD (Schema.org)
